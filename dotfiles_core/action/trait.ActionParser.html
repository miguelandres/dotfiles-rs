Builds a single action of type ActionParser::ActionType from Yaml tree object that represents the action’s configuration and a default settings object.

Returns an Error containing a human readable string in case there was an issue building the action.

Builds a list of actions of type ActionParser::ActionType from Yaml tree object that represents the actions’ configurations and a default settings object.

Returns an Error containing a human readable string in case there was an issue building the action.

The default implementation assumes there must be Yaml array whose items each represent an individual action
